Data Analysis & Visualisation: Manufacturing Dataset

Data analysis in manufacturing serves as a crucial part for decision-making, offering a clear view of the entire production lifecycle. It enables manufacturers to make data driven decisions about resource allocations, process improvements, and quality control. Furthermore, by analyzing data from various stages of manufacturing, companies can anticipate equipment failures, streamline supply chain management, and reduce downtime, thereby driving cost savings and boosting overall efficiency.

This project involves an exploratory data analysis (EDA) of a dataset, which records the number of defects in a manufacturing or production process over a 10-day period. The primary focus of this analysis is to identify insights that could be crucial for process improvement and quality assurance. I focused on an analysis using Python and libraries like Pandas as well as Matplotlib and Seaborn for data visualisation and exploration. This project is a representation of my beginnings in data analytics, so there's more to come in the future!
