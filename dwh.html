Data Warehouse Architecture

In the domain of manufacturing, data serves as the keystone for informed decision-making. My project revolves around constructing a data warehouse architecture that integrates real-time IoT sensor data, production metrics, supply chain information, and quality assurance data. The core objective was to create a platform that empowers stakeholders to harness data for operational optimization, quality enhancement, and supply chain agility.

This project is a showcase of my journey in designing a data architecture tailored for manufacturing. This project is a reflection of my interests in the domain of data engineering and architecture, where I delve into the intricacies of handling data landscapes typical to modern production environments.
